# Release History

## 0.4.0 (2026-01-12)
### Breaking Changes

- Type of `MonetaryCommitment.Name` has been changed from `*OfferTermInfo` to `*OfferTermInfoName`
- Type of `MonetaryCredit.Name` has been changed from `*OfferTermInfo` to `*OfferTermInfoName`
- Type of `RecurringCharge.Name` has been changed from `*OfferTermInfo` to `*OfferTermInfoName`
- Type of `ResourceRateCardInfo.OfferTerms` has been changed from `[]OfferTermInfoAutoGeneratedClassification` to `[]OfferTermInfoClassification`
- Enum `OfferTermInfo` has been removed
- Function `*MonetaryCommitment.GetOfferTermInfoAutoGenerated` has been removed
- Function `*MonetaryCredit.GetOfferTermInfoAutoGenerated` has been removed
- Function `*OfferTermInfoAutoGenerated.GetOfferTermInfoAutoGenerated` has been removed
- Function `*RecurringCharge.GetOfferTermInfoAutoGenerated` has been removed
- Struct `ErrorResponse` has been removed
- Struct `InfoField` has been removed
- Struct `RateCardQueryParameters` has been removed

### Features Added

- Type of `UsageSample.InfoFields` has been changed from `*InfoField` to `any`
- New enum type `OfferTermInfoName` with values `OfferTermInfoNameMonetaryCommitment`, `OfferTermInfoNameMonetaryCredit`, `OfferTermInfoNameRecurringCharge`
- New function `*MonetaryCommitment.GetOfferTermInfo() *OfferTermInfo`
- New function `*MonetaryCredit.GetOfferTermInfo() *OfferTermInfo`
- New function `*OfferTermInfo.GetOfferTermInfo() *OfferTermInfo`
- New function `*RecurringCharge.GetOfferTermInfo() *OfferTermInfo`


## 0.3.0 (2023-11-24)
### Features Added

- Support for test fakes and OpenTelemetry trace spans.


## 0.2.0 (2023-03-28)
### Features Added

- New struct `ClientFactory` which is a client factory used to create any client in this module
