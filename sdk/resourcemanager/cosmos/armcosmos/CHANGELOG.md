# Release History

## 4.0.0-beta.3 (2025-11-14)
### Breaking Changes

- Function `*CassandraClustersClient.BeginCreateUpdate` parameter(s) have been changed from `(context.Context, string, string, ClusterResource, *CassandraClustersClientBeginCreateUpdateOptions)` to `(context.Context, string, string, ClusterResourceAutoGenerated, *CassandraClustersClientBeginCreateUpdateOptions)`
- Function `*CassandraClustersClient.BeginUpdate` parameter(s) have been changed from `(context.Context, string, string, ClusterResource, *CassandraClustersClientBeginUpdateOptions)` to `(context.Context, string, string, ClusterResourceAutoGenerated, *CassandraClustersClientBeginUpdateOptions)`
- Type of `ClusterResourceProperties.ProvisionError` has been changed from `*CassandraError` to `*ErrorDetailAutoGenerated`
- Type of `ClusterResourceProperties.ProvisioningState` has been changed from `*ManagedCassandraProvisioningState` to `*GarnetCacheProvisioningState`
- Type of `DataTransferJobProperties.Error` has been changed from `*ErrorResponseAutoGenerated` to `*ErrorResponseAutoGenerated2`
- Type of `SQLRoleDefinitionResource.Permissions` has been changed from `[]*Permission` to `[]*PermissionAutoGenerated`
- Type of `TableRoleDefinitionResourceProperties.Permissions` has been changed from `[]*PermissionAutoGenerated` to `[]*Permission`
- Field `ClusterResource` of struct `CassandraClustersClientCreateUpdateResponse` has been removed
- Field `ClusterResource` of struct `CassandraClustersClientGetResponse` has been removed
- Field `ListClusters` of struct `CassandraClustersClientListByResourceGroupResponse` has been removed
- Field `ListClusters` of struct `CassandraClustersClientListBySubscriptionResponse` has been removed
- Field `ClusterResource` of struct `CassandraClustersClientUpdateResponse` has been removed
- Field `Identity` of struct `ClusterResource` has been removed
- Field `AuthenticationMethod`, `AutoReplicate`, `AzureConnectionMethod`, `BackupSchedules`, `CassandraAuditLoggingEnabled`, `CassandraVersion`, `ClientCertificates`, `ClusterNameOverride`, `Deallocated`, `DelegatedManagementSubnetID`, `ExternalDataCenters`, `ExternalGossipCertificates`, `ExternalSeedNodes`, `GossipCertificates`, `HoursBetweenBackups`, `InitialCassandraAdminPassword`, `PrivateLinkResourceID`, `PrometheusEndpoint`, `RepairEnabled`, `RestoreFromBackupID`, `ScheduledEventStrategy`, `SeedNodes` of struct `ClusterResourceProperties` has been removed
- Field `Code`, `Message` of struct `ErrorResponseAutoGenerated` has been removed
- Field `ID` of struct `PermissionAutoGenerated` has been removed
- Field `SystemData` of struct `PrivateEndpointConnection` has been removed
