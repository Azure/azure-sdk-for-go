# Release History

## 0.1.0 (Unreleased)

### Features Added

* Initial preview release of the Azure File Shares management library for Go.
* Support for managing Azure File Shares, including creating, updating, listing, and deleting shares.
* Operations for managing share snapshots, including listing and deleting snapshots.
* Informational operations to retrieve share usage, quotas, and related service limits.
* Support for configuring share properties such as metadata and share-level settings.
